# Pruizlezcano Tap

## How do I install these formulae?

`brew install pruizlezcano/tap/<formula>`

Or `brew tap pruizlezcano/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "pruizlezcano/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
